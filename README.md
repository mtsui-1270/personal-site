# Mariah Tsui — Personal Website


## Purpose
Built as a creative and minimal space to express myself and present my work, background,
interests while staying focused on clean design and personal touches over complexity.


## Live Site
[mariaht.vercel.app](https://mariaht.vercel.app)

## Built from Scratch With
- **HTML** -> structure and content
- **CSS** -> minimal styling, custom animations, flexbox layout
- **Vanilla JavaScript** -> typewriter effect, time-based greeting, live Spotify API Fetch
- **Vercel** -> (backend + deployment)
- **Spotify API** -> (currently playing widget)

## Pages
- **About** — bio, socials, currently section
- **Projects** — current and past projects
- **Awards** — scholarships and recognitions

## Features
- Typewriter effect in English and Chinese
- Live Spotify now playing widget
- Time-based greeting
